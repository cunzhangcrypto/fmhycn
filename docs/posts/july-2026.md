# 维基更新

- 在 [Flow Launcher](https://www.flowlauncher.com) 中添加了 [FMHY 插件](https://www.flowlauncher.com/plugins/fmhy-search-3f7a2b1c-9e4d-4f8a-b3c2-1a5d8e7f0b9c)。感谢 @finn 制作了这个插件。

- 在阅读部分添加了 [Kindle 工具](https://fmhy.net/reading#kindle-tools)小节。感谢 @Dan 和 @Kiwi Kitten 制作了这一部分。

- 在流媒体部分添加了 [P-Stream 分支](https://fmhy.net/video#p-stream-forks)小节。

- 在互联网工具部分添加了 [Chromium 工具](https://fmhy.net/internet-tools#chromium-tools)小节。

- 在人文学科部分添加了 [世界数据](https://fmhy.net/educational#world-data)小节。感谢 @hiz 提出的想法。

- 在人文学科部分添加了 [旗帜学 / 旗帜](https://fmhy.net/educational#vexillology-flags)小节。感谢 @MTG 收集资源。

- 将 [航空航天工程](https://fmhy.net/educational#aerospace-engineering) 细分为子小节：[火箭学](https://fmhy.net/educational#rocketry) / [无人机](https://fmhy.net/educational#drones) / [模拟器](https://fmhy.net/educational#simulators)。感谢 @yeet 收集网站 + @tomato 提出的想法。

- 重新组织了 [视频播放器](https://fmhy.net/video-tools#video-players) 部分，并将 MPV 和 MPC-HC 都移至 MPC-QT 之上，因为人们觉得其易用性并不能抵消其潜在问题或缺失的功能（如对配置的支持）。[修改前后对比](https://i.ibb.co/ksvk9R6C/1231234234.png) / [2](https://i.imgur.com/HBp3zGh.png)

- 为 [邮件客户端](https://fmhy.net/internet-tools#email-clients) 添加了平台图标，感谢 @draven 完成了这项工作。[修改前后对比](https://i.ibb.co/8gCfmYnF/354756uj.png) / [2](https://i.imgur.com/ATMzFBm.png)

- 改进了 [博文页面](https://fmhy.net/posts) 的用户界面，感谢 @cufiy 完成了这项工作。[修改前后对比](https://i.ibb.co/dJQxgMk8/image.png) / [2](https://i.imgur.com/CzMIKFG.png)

- 新增了 2 个 FMHY 重复链接检查器：[Link Checker](https://link-checker-215e95.gitlab.io/) + [FMHY Check](https://fmhycheck.vercel.app/)，感谢 @cufiy 和 @casmay 制作了这些工具。

# 新增星标 ⭐

- 为 Android YouTube Music 应用中的 [ArchiveTune](https://fmhy.net/mobile#youtube-music) 添加星标，其界面美观，且对用户来说似乎运行良好。

- 为食品部分的 [Too Good To Go](https://fmhy.net/misc#food) 添加星标，这是获取廉价食品并防止食物浪费的好方法，对我们的试用成员来说效果不错。

- 为阅读追踪/数据库中的 [Hardcover](https://fmhy.net/reading#tracking-database) 添加星标，其界面美观，功能丰富，并能从其他平台（如 goodreads）导入数据。

- 为粘贴箱中的 [Yasosu](https://fmhy.net/text-tools#pastebins) 添加星标，支持 Markdown，允许编辑帖子，并具有自定义 URL。

- 为精选推荐中的 [The Criterion Closet](https://fmhy.net/video#curated-recommendations) 添加星标，这是一个带有筛选功能的 3D Criterion 步入式衣柜重现。

- 为 Linux 音频中的 [LibrePods](https://fmhy.net/linux-macos#linux-audio) 添加星标，允许你在 Linux（和已 root 的 Android）上使用 AirPod 功能。

- 为动漫种子下载中的 [SeaDex](https://fmhy.net/video#anime-torrenting) 添加星标，这是一个动漫视频质量指南，可与 Stremio + AIOStream 等工具结合使用。

- 为 Linux 系统中的 [LACT](https://fmhy.net/linux-macos#linux-system) 添加星标，这是一个功能丰富的 GPU 配置 + 监控工具，简化了超频操作。

- 为机器学习中的 [Unsloth](https://fmhy.net/ai#machine-learning) 添加星标，这是一个易于使用的微调 + AI 套件，似乎很受欢迎。

- 为元素周期表中的 [Zperiod](https://fmhy.net/educational#periodic-tables) 添加星标，这是一个带有 3D 原子模型的简洁交互式元素周期表。

- [Bookracy](https://fmhy.net/reading#ebooks) 已恢复，已在电子书部分重新添加并加星。

- [Soft98 和 DownloadHa](https://fmhy.net/downloading#software-sites) 都已再次可用，已在软件部分重新添加并加星。
